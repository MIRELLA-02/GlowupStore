# 💄 Glow Up Store

Uma aplicação Full Stack desenvolvida para gerenciamento e exibição de produtos de uma loja de cosméticos. O sistema integra backend + frontend, permitindo cadastrar, listar e visualizar produtos de forma prática e organizada.

--------------

# ✨ Sobre o projeto

A Glow Up Store foi criada com o objetivo de simular um e-commerce de cosméticos, focando na integração entre front-end e back-end.

O sistema permite:

- Cadastro de produtos
- Upload de imagens
- Listagem dinâmica de itens
- Organização por categorias e marcas.
